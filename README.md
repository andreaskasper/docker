# docker
Some easy to deploy docker examples

## Teamspeak
Creates a simple Teamspeak3 Server via Docker and runs it on port 9987
Folder: /teamspeak

You can get your token for administration via docker logs <NAME_OF_DOCKERCONTAINER>
